Que es

Fuzzy and Friends para RD. Plataforma educativa animada. Ninos exploran isla interactiva basada en geografia real de RD. Fuzzy el solenodonte es protagonista. Amigos por crear.

Formato Episodico

Cada episodio = paquete independiente:

  • Video 3-5 min (AnimaClaw)
  • 4-5 juegos tematicos
  • PDF Where's Waldo, postales, periodico retro
  • Datos curiosos
